# 📅 SecuDeck 스케줄 봇 커맨드 목록

## 기본 사용법
모든 커맨드는 `/일정` 으로 시작합니다.  
디스코드 채팅창에 `/일정` 을 입력하면 자동완성 목록이 뜹니다.

---

## 커맨드 상세

### 📌 `/일정 등록`
새 일정을 등록합니다.

| 파라미터 | 필수 여부 | 형식 | 예시 |
|---|---|---|---|
| `제목` | 필수 | 텍스트 | `팀 회의` |
| `날짜` | 필수 | `YYYY-MM-DD` | `2026-04-10` |
| `시간` | 선택 | `HH:MM` | `14:30` |
| `설명` | 선택 | 텍스트 | `주간 보안 점검 회의` |

**예시**
```
/일정 등록 제목:팀 회의 날짜:2026-04-10 시간:14:30 설명:주간 보안 점검
```

---

### 📋 `/일정 목록`
앞으로 예정된 일정을 최대 10개 보여줍니다.

```
/일정 목록
```

---

### 📅 `/일정 오늘`
오늘 날짜의 일정만 조회합니다.

```
/일정 오늘
```

---

### 📆 `/일정 이번주`
오늘부터 이번 주 일요일까지의 일정을 조회합니다.

```
/일정 이번주
```

---

### 🔍 `/일정 날짜검색`
특정 날짜의 일정을 조회합니다.

| 파라미터 | 필수 여부 | 형식 | 예시 |
|---|---|---|---|
| `날짜` | 필수 | `YYYY-MM-DD` | `2026-04-15` |

**예시**
```
/일정 날짜검색 날짜:2026-04-15
```

---

### 🔎 `/일정 상세`
일정 ID로 상세 정보를 확인합니다.  
ID는 `/일정 목록` 에서 대괄호 안의 숫자입니다. (예: `[3]`)

| 파라미터 | 필수 여부 | 형식 | 예시 |
|---|---|---|---|
| `id` | 필수 | 숫자 | `3` |

**예시**
```
/일정 상세 id:3
```

---

### ✏️ `/일정 수정`
등록된 일정을 수정합니다. 수정할 항목만 입력하면 됩니다.

| 파라미터 | 필수 여부 | 형식 | 예시 |
|---|---|---|---|
| `id` | 필수 | 숫자 | `3` |
| `제목` | 선택 | 텍스트 | `긴급 보안 회의` |
| `날짜` | 선택 | `YYYY-MM-DD` | `2026-04-11` |
| `시간` | 선택 | `HH:MM` | `15:00` |
| `설명` | 선택 | 텍스트 | `일정 변경됨` |

**예시** — 시간만 바꾸고 싶을 때
```
/일정 수정 id:3 시간:15:00
```

---

### 🗑️ `/일정 삭제`
일정 ID로 일정을 삭제합니다.

| 파라미터 | 필수 여부 | 형식 | 예시 |
|---|---|---|---|
| `id` | 필수 | 숫자 | `3` |

**예시**
```
/일정 삭제 id:3
```

---

### 🔔 `/일정 알림채널` *(관리자 전용)*
일정 시작 30분 전에 자동 알림을 받을 채널을 설정합니다.  
시간이 등록된 일정에만 알림이 발송됩니다.

| 파라미터 | 필수 여부 | 형식 | 예시 |
|---|---|---|---|
| `채널` | 필수 | 채널 멘션 | `#일정-알림` |

**예시**
```
/일정 알림채널 채널:#일정-알림
```

---

## 주의사항

- 날짜는 반드시 `YYYY-MM-DD` 형식으로 입력하세요. (예: `2026-04-10`)
- 시간은 24시간제 `HH:MM` 형식으로 입력하세요. (예: `14:30`)
- 30분 전 자동 알림은 **시간이 등록된 일정**에만 발송됩니다.
- 알림 채널 설정은 **서버 관리 권한**이 있는 멤버만 가능합니다.
